## Affected Pages/Routes
- `/` (AI Advisor home) — briefing card renders, FAB opens chat sheet, metric cards show data
- `/dashboard` — OML gauge, pillar bars, activity log populate from SQLite
- `/academics` — course list CRUD, GPA calculation, "Simulate GPA" flow
- `/fitness` — 6 ACFT event cards, score entry, trend chart
- `/leadership` — role cards, CST score, badge display
- `/onboarding/*` — 5-step flow gates first launch, data persists to SQLite

## Key Interactions to Verify
- Enter AFT score on Fitness tab → OML recalculates → AI Advisor briefing updates
- Tap FAB on AI Advisor → chat sheet opens → send message → streaming response
- Complete onboarding → redirects to AI Advisor with first insight
- Toggle airplane mode → data entry works, AI shows cached briefing, query queues
- What-if: adjust slider → OML projection updates in real-time

## Edge Cases
- First launch with zero data → onboarding gate triggers
- GPA input of 5.0 → validation rejects inline
- AFT alternate event (Leg Tuck instead of Plank) → engine handles correctly
- Network failure mid-AI-response → partial response + retry chip
- All scores at maximum (4.0 GPA, 600 AFT, 100 leadership) → OML ceiling
- 47-character course name → doesn't overflow course list cards

## Critical Paths
- Onboarding → first insight → daily return (core retention loop)
- Data entry → OML recalculation → AI micro-insight (core value loop)
- Offline data entry → reconnect → queue flush → AI responds (field training scenario)
