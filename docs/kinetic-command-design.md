## Problem Statement

The current "Modern Vanguard" design system (light mode, olive/gold) served well for the v1 build but doesn't match the aspirational "tactical command center" experience. The new "Kinetic Command" system transforms the app into a dark-first HUD aesthetic with purple/gold accents, glow effects, and editorial typography.

## What's Changing

| Element | Modern Vanguard (current) | Kinetic Command (new) |
|---------|--------------------------|----------------------|
| Default mode | Light | Dark |
| Primary | #343c0a (olive) | #d9b9ff (purple) |
| Primary container | #4b5320 | #450084 (deep purple) |
| Secondary | #6d5d2f (brown) | #dbc585 (Duke Gold) |
| Tertiary | #735c00 (gold) | #c3cc8c (Tactical Olive) |
| Surface | #f5faff (blue-white) | #151317 (void black) |
| Label font | Inter | Space Grotesk |
| Shadows | Ambient (black-based) | Glow Drops (colored, from within) |
| Borders | Ghost borders (outline_variant 20%) | Ghost borders (outline_variant 15%) |
| Corner radius | rounded-md/lg | rounded-sm (sharper, military) |

## Approach: Token-First Migration

### Step 1: Update DESIGN.md
Replace with the full "Kinetic Command & Editorial Depth" specification.

### Step 2: Update dark tokens
Update `src/theme/darkTokens.ts` (or the dark colors in tokens.ts) to match the Kinetic Command palette exactly from the spec.

### Step 3: Make dark mode default
In `src/theme/ThemeContext.tsx`, change the default mode from 'light' to 'dark'.

### Step 4: Install Space Grotesk font
Add Space Grotesk to the app for label typography. Update the typography tokens.

### Step 5: Update component styles
For each component, add:
- Glow Drop shadows (colored shadows using primary_container or secondary_container at 15% opacity, 30px blur)
- Sharper corners (roundness.sm = 0.125rem for buttons, roundness.lg for cards)
- Glassmorphism updates (surface_variant at 60% opacity + heavier blur)

### Step 6: Update screens
Each screen reads from ThemeContext (already set up in Sprint 2). The dark palette flows through automatically. Screen-specific adjustments for glow effects and glassmorphism.
