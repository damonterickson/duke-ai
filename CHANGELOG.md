# Changelog

All notable changes to Duke Vanguard will be documented in this file.

## [0.2.0.0] - 2026-03-24

### Added
- Upload & Sync screen with document type cards and sync status
- Canvas Integration screen with connection flow and course list
- ACFT Log screen with event-by-event scoring form and history
- Intelligence Brief screen with OML breakdown, AI analysis, and action items
- Mission Ready onboarding completion screen with animated checkmark

### Changed
- Migrated fitness, academics, and leadership tabs to dual-theme system (Modern Vanguard + Kinetic Command)
- Migrated all 7 onboarding screens to dual-theme system
- Made all UI buttons and interactive elements functional (zero dead buttons)
- Fixed web magic link redirect to use localhost URL on web platform

## [0.1.0.0] - 2026-03-21

### Added
- Initial app with Mission, Intel, Squad, and Profile tabs
- OML calculator engine with deterministic scoring
- AI Advisor with OpenRouter integration
- Onboarding flow with year group, GPA, AFT, leadership, and branch selection
- Dual-theme design system (Modern Vanguard light + Kinetic Command dark)
- Offline-first architecture with SQLite + AsyncStorage
